# DEV_COMMANDS.md (Recommended)
Define the standard commands agents must use.

## Backend
- install: pnpm i
- dev: pnpm --filter api dev
- test: pnpm --filter api test
- lint: pnpm --filter api lint

## Shared
- test: pnpm --filter shared test

## Mobile
- dev: pnpm --filter mobile start
- test: pnpm --filter mobile test
